## 构造格雷码


```c++
int g(int n) { return n ^ (n >> 1); }
```

## 格雷码构造原数（逆变换）

```c++
int rev_g(int g) {
  int n = 0;
  for (; g; g >>= 1) n ^= g;
  return n;
}
```
