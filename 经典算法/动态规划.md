## 子数组最大累加和

```python
dp = [-inf] * (n + 1)
dp[1] = a[1]
for i in range(2, n + 1):
    dp[i] = max(0, dp[i - 1]) + a[i]
print(max(dp))
```

## 最长上升子序列（LIS）

DP解法 `dp[i] = 以第i个数结尾的最长上升子序列的长度`

```python
dp = [1] * (n + 1)  
for i in range(1, n + 1):
    for j in range(1, i):
        if a[i] > a[j]:
            dp[i] = max(dp[i], dp[j] + 1)
```

**二分 + 贪心**  `O(nlogn)
`
[P3902 递增 - 洛谷](https://www.luogu.com.cn/problem/P3902) 

[P1020 [NOIP 1999 提高组] 导弹拦截 - 洛谷](https://www.luogu.com.cn/problem/P1020)

```c++
void solve() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) cin >> a[i];
    vector<int> dp;
    for (int x : a) {
        auto it = lower_bound(dp.begin(), dp.end(), x); // 非递减用upper_bound
        if (it == dp.end()) dp.push_back(x);
        else *it = x;
    }
    cout << dp.size() << "\n";
}
```



## 最长公共子序列（LCS）

`dp[i][j]:A的前i个元素和序列B的前j个元素的最长公共子序列长度` $O(n^2)$

```python
for i in range(1, n + 1):
    for j in range(1, m + 1):
        if a[i - 1] == b[j - 1]:
            dp[i][j] = dp[i - 1][j - 1] + 1
        else:
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
```

**如果A和B都是1-n的排列 可以将LCS转化为LIS**


## 最长回文子序列（LPS）

`dp[i][j]` 表示字符串 `s[i..j]` 的最长回文子序列长度

```python
for i in range(n):
    dp[i][i] = 1
for length in range(2, n + 1):
    for i in range(n - length + 1):
        j = i + length - 1
        if s[i] == s[j]:
            dp[i][j] = dp[i + 1][j - 1] + 2
        else:
            dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])
```
