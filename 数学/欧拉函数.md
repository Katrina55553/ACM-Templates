
## 欧拉函数

欧拉函数$\phi(n)$ 表示小于等于 n 的正整数中与 n 互质的数的个数。即：
$\phi(n) = \left| \{ k \in \mathbb{Z}^+ \mid 1 \leq k \leq n, \gcd(k, n) = 1 \} \right|$
$\text{当 }n\text{ 是质数的时候,显然有 }\varphi(n)=n-1\text{。}$
$\text{ 若 }n=p^k\text{,其中 }p\text{ 是质数,那么 }\varphi(n)=p^k-p^{k-1}。$

**求一个数的欧拉函数值，在质因数分解的同时求：**

```python
 def euler_phi(n):  
    if n == 1:  
        return 1  
    else:  
        phi = n  
        max_n = int(sqrt(n))  
        for i in range(2, max_n + 1):  
            if n % i == 0:  
                phi -= phi // i  
                while n % i == 0:  
                    n //= i  
        if n > 1:  
            phi -= phi // n  
        return phi
```

**筛法求欧拉函数：**

```python
def euler_phi_sieve(n):
    phi = list(range(n + 1))  # 初始化phi[i] = i
    for i in range(2, n + 1):
        if phi[i] == i:
            for j in range(i, n + 1, i):  # 遇到质数i 调整i所有倍数的欧拉函数值
                phi[j] -= phi[j] // i
    return phi
    
```

## 欧拉降幂

幂次大于欧拉函数值 --> 欧拉降幂

$a^b\equiv\begin{cases}a^{b\mathrm{~mod~}\varphi(m)},&\gcd(a,m)=1,\\a^b,&\gcd(a,m)\neq1,b<\varphi(m),&(\mathrm{mod~}m)\\a^{(b\mathrm{~mod~}\varphi(m))+\varphi(m)},&\gcd(a,m)\neq1,b\geq\varphi(m).&\end{cases}$

```python
a, m, b = input().split()  # 读取为字符串（b非常大）
a, m = int(a), int(m)
def euler_phi(n):
    if n == 1:
        return 1
    max_n = int(sqrt(n))
    phi = n
    for i in range(2, max_n + 1):
        if n % i == 0:
            phi -= phi // i
            while n % i == 0:
                n //= i
    if n > 1:
        phi -= phi // n
    return phi

if m == 1:
    exit(print(0))
phi = euler_phi(m)
flg = 0
b_mod_phi = 0
for digit in b:
    b_mod_phi = (b_mod_phi * 10 + int(digit)) % phi

if len(b) > len(str(phi)):
    flg = 1
elif len(b) == len(str(phi)):
    flg = b >= str(phi)

if gcd(a, m) == 1:
    int_b = b_mod_phi
else:
    if flg:
        int_b = b_mod_phi + phi
    else:
        int_b = int(b)

print(pow(a, int_b, m))
```
